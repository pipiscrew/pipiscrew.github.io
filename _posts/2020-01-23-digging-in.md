> update old version to newest

1. wp-includes/version.php should tell you exactly what the WP version is/was.
2. Safety step. You should install first the same version ([https://wordpress.org/download/release-archive/](https://wordpress.org/download/release-archive/)) and make sure the blog works.
3. Then follow the instructions for [Upgrading_WordPress](http://codex.wordpress.org/Upgrading_WordPress)
