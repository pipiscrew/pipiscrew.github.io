Adminer (formerly [phpMinAdmin](http://sourceforge.net/projects/phpminadmin/)) is a full-featured database management tool written in PHP. Conversely to phpMyAdmin, it consist of a single file ready to deploy to the target server. It is available for MySQL, PostgreSQL, SQLite, MS SQL and Oracle.

[http://www.adminer.org/](http://www.adminer.org/)

[http://sourceforge.net/projects/adminer/](http://sourceforge.net/projects/adminer/)
