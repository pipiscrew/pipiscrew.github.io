### File formats, applications and converting PDF to AZW3 or mobi

file formats :

**mobi** is Mobipocket e-book format is based on the Open eBook standard using XHTML
**AZW** files debuted with the first Amazon Kindle in 2007
**AZW3** supports a subset of HTML5 and CSS3 features, AZW3 is epub with an amazon wrapper, introduted in 2011
