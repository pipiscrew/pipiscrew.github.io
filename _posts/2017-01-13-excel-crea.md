Use this function on first cell, then drag the cell, it will concatenate all rows. Copy the cells to a text file, save as csv.

```js

=CONCATENATE(A2,"|",B2,"|",C2,"|",D2,"|",E2,"|",F2,"|",G2,"|",H2,"|",I2,"|",J2,"|",K2,"|",L2,"|",M2)

```
