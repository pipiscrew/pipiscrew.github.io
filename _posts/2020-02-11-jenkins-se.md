Jenkins, an open source server used to perform automated tasks, can be abused to launch distributed denial of service (DDoS) attacks. DDoS attacks are possible because of a vulnerability in the Jenkins codebase. The bug (tracked as CVE-2020-2100) has been fixed in Jenkins v2.
